# IBM HR Analytics - Employee Attrition Analysis

## 概要

IBM HR Analytics Employee Attrition & Performance データセットを用いて、
従業員の離職状況を分析したSQL・Tableauのポートフォリオです。

---

## 使用技術

- SQL(BigQuery)
- Tableau
- GitHub

---

## 分析目的

以下の観点から離職状況を分析しました。

- 全体の離職状況の把握
- 部署・職種ごとの離職率比較
- 社員属性と離職率の比較
- 給与・勤続年数と離職率の比較
- 勤務環境・満足度と離職率の比較
- 複数条件を組み合わせた離職傾向の分析
