# IBM HR Analytics - Employee Attrition Analysis

## 概要

IBM HR Analytics Employee Attrition & Performance データセットを用いて、
従業員の離職状況をSQLとTableauで分析したポートフォリオです。

---

## 分析目的

従業員の離職状況を把握し、部署・職種・社員属性・給与・勤務状況などの観点から、離職率との関連性を分析することを目的としています。

本分析では、各項目を単独で比較するだけでなく、複数の条件を組み合わせることで、離職率が高い傾向にある社員層を確認します。

---

| 技術 | 用途 |

## 分析内容

本プロジェクトでは、全体の離職状況を確認したうえで、部署・職種、社員属性、待遇、勤務状況の順に分析を進めます。

| ファイル | 分析内容 |
|----------|----------|
| `01_basic.sql` | 全社員数・離職者数・在籍者数・全体離職率の確認 |
| `02_department.sql` | 部署・職種ごとの社員数と離職率の比較 |
| `03_employee.sql` | 年代・性別・婚姻状況・出張頻度など社員属性ごとの離職率比較 |
| `04_salary.sql` | 給与・勤続年数・昇進状況など待遇面と離職率の分析 |
| `05_workstyle.sql` | 残業・満足度・ワークライフバランスなど勤務状況と離職率の分析 |
| `06_summary.sql` | 複数条件を組み合わせた離職傾向の分析 |
| `07_tableau.sql` | Tableauで使用する可視化用データセットの作成 |

## 分析結果

### 1. 全体の離職状況

全社員1,470名のうち237名が離職しており、全体の離職率は16.12%でした。

### 2. 部署・職種別の傾向

部署別では、Sales部門の離職率が20.63%と最も高く、446名中92名が離職していました。

職種別では、Sales Representativeの離職率が39.76%と最も高く、83名中33名が離職していました。

### 3. 社員属性別の傾向

年代別では、20代以下の離職率が27.91%と最も高い結果となりました。

婚姻状況別では、Singleの離職率が25.53%と最も高く、出張頻度別ではTravel_Frequentlyの離職率が24.91%と最も高い結果となりました。

### 4. 給与・勤続年数別の傾向

離職者の平均月収は4,787.09、在籍者の平均月収は6,832.74であり、離職者の方が平均月収が低い傾向が確認できました。

給与帯別では3,000未満の離職率が28.61%、勤続年数別では3年未満の離職率が29.82%と、それぞれ最も高い結果となりました。

### 5. 働き方・満足度別の傾向

残業ありの社員の離職率は30.53%で、残業なしの社員の10.44%を大きく上回りました。

仕事満足度別では、最低評価であるJobSatisfaction「1」の離職率が22.84%と最も高い結果となりました。

### 6. 複数条件を組み合わせた分析

部署・職種・残業有無を組み合わせて分析した結果、以下の社員セグメントの離職率が最も高くなりました。

- 部署：Sales
- 職種：Sales Representative
- 残業：あり
- 社員数：24名
- 離職者数：16名
- 離職率：66.67%

単一項目だけでなく複数の条件を組み合わせることで、特に離職率が高い社員層を確認できました。

---

## 考察

分析結果から、離職率の高さには以下の項目が関連している可能性があります。

- Sales部門およびSales Representative
- 残業あり
- 勤続3年未満
- 月収3,000未満
- 20代以下
- 独身
- 頻繁な出張
- 仕事満足度が低い

特に、Sales部門のSales Representativeで残業を行っている社員は、離職率が66.67%と高い結果になりました。

そのため、離職防止策を検討する場合は、入社初期の営業担当者に対する業務量の確認、残業状況の改善、待遇やフォロー体制の見直しなどが優先的な検討対象になると考えられます。
